  <files>scripts/backtest.py</files>
  <action>
    Create `scripts/backtest.py` with:

    **CLI args (argparse):**
    - `--model` (str, required): Path to pre-trained XGBoost .pkl file
    - `--btc` (str, default="data/BTCUSDT_4h.parquet"): BTC Parquet path
    - `--eth` (str, default="data/ETHUSDT_4h.parquet"): ETH Parquet path
    - `--sol` (str, default="data/SOLUSDT_4h.parquet"): SOL Parquet path
    - `--start` (str, default=None): Start date filter, e.g. "2024-01-01"
    - `--end` (str, default=None): End date filter, e.g. "2024-12-31"
    - `--threshold` (float, default=0.6): Probability threshold for BUY signal; SELL threshold = 1.0 - threshold
    - `--fee-bps` (int, default=0): Transaction cost in basis points (0 = free)
    - `--long-only` (bool, store_true, default=False): If set, ignore SELL signals (just exit on SELL, don't go short); default is long-only anyway since short not implemented, but keep the flag for future extension
    - `--output` (str, default=None): If set, write stats report as CSV to this path

    **Function `prepare_features(btc_path, eth_path, sol_path, start=None, end=None) -> pd.DataFrame`:**
    1. Load each Parquet: `pd.read_parquet(path)`
    2. Ensure DatetimeIndex: `df.index = pd.to_datetime(df.index)` if not already DatetimeIndex
    3. Ensure columns are lowercase: `df.columns = df.columns.str.lower()`
    4. Call pipeline in EXACT order (critical — from Phase 04-03):
       ```python
       from bot.data.features import compute_features, compute_cross_asset_features
       feat = compute_features(btc)
       feat = compute_cross_asset_features(feat, {"ETH/USD": eth, "SOL/USD": sol})
       feat = feat.dropna()
       ```
    5. Filter by date range if --start/--end provided:
       ```python
       if start: feat = feat[feat.index >= pd.Timestamp(start, tz="UTC")]
       if end:   feat = feat[feat.index <= pd.Timestamp(end, tz="UTC")]
       ```
    6. Return feat

    Add a stub `main()` that just calls `prepare_features()` and prints shape for now:
    ```python
    if __name__ == "__main__":
        args = parse_args()
        feat = prepare_features(args.btc, args.eth, args.sol, args.start, args.end)
        print(f"Feature matrix: {feat.shape[0]} bars × {feat.shape[1]} columns")
        print(f"Date range: {feat.index[0]} → {feat.index[-1]}")
    ```

    Import only what's needed: argparse, pandas, from bot.data.features import ....
    Do NOT import xgboost or quantstats yet (those are in 10-02 and 10-03).
