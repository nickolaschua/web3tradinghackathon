```python
class StateManager:
    def __init__(self, path: Path | str = _DEFAULT_PATH) -> None:
        self.path = Path(path)
    def write(self, state: dict[str, Any]) -> None:
        try:
            tmp = self.path.with_suffix(".tmp")
            with open(tmp, "w") as f:
                json.dump(state, f, indent=2, default=str)
            os.replace(tmp, self.path)
        except Exception as exc:
            logger.error("StateManager.write() failed: %s", exc)
    def read(self) -> dict[str, Any]:
        try:
            if self.path.exists():
                with open(self.path) as f:
                    return json.load(f)
        except Exception as exc:
            logger.warning("StateManager.read() failed, returning empty state: %s", exc)
        return {}
```

### Task 1: Implement TelegramAlerter real HTTP send

**File:** `bot/monitoring/telegram.py`

Replace the stub with a real implementation. Requirements:

1. **Module constants:**
   ```python
   API_URL = "https://api.telegram.org/bot{token}/sendMessage"
   TIMEOUT_SECONDS = 5
   LEVEL_EMOJIS = {
       "INFO": "ℹ️",
       "WARN": "⚠️",
       "ERROR": "🔴",
       "CRITICAL": "🚨",
   }
   ```

2. **`__init__(self, token: str, chat_id: str) -> None`:**
   - Store `self.token` and `self.chat_id`
   - Set `self._enabled = bool(token and chat_id)` — False if either is empty/None
   - If not enabled, log WARNING once: `"TelegramAlerter disabled: token or chat_id missing"`

3. **`send(self, message: str, level: str = "INFO") -> bool`:**
   - If `not self._enabled`: log at DEBUG `"[Telegram disabled] %s"` and return `False`
   - Build `text = f"{LEVEL_EMOJIS.get(level, '')} {message}"`
   - POST to `API_URL.format(token=self.token)` with payload:
     ```python
     {"chat_id": self.chat_id, "text": text, "parse_mode": "HTML"}
     ```
   - Use `timeout=TIMEOUT_SECONDS`
   - Wrap entire method in `try/except Exception` — on ANY exception: log at DEBUG, return `False`
   - On HTTP success: return `True`
   - Response status check: `response.raise_for_status()` inside the try block

4. **Never raise** — the `try/except Exception: return False` wrapper is the outermost block of `send()`. Trading loop must never see an exception from this class.

5. **Import `requests`** at top of file.

**Verification:** After writing, mentally trace: if token is empty string, `_enabled=False`, send() returns False without network call. If requests.post raises ConnectionError, except catches it, returns False. main.py calls `telegram.send("msg")` with no level arg — works because default is "INFO".

### Task 2: Harden StateManager

**File:** `bot/persistence/state_manager.py`

Enrich the existing stub. The atomic write core (`os.replace`) stays; add metadata injection, backup, fallback read, age check, and JSON serializer.

1. **Add imports:** `import shutil`, `from datetime import datetime, timezone`, `from dataclasses import asdict`, `import dataclasses`

2. **Module constant:** `VERSION = 1`

3. **`__init__`:** Add two derived attributes after `self.path = Path(path)`:
   ```python
   self.tmp_file = self.path.with_suffix(".tmp")
   self.backup_file = self.path.with_suffix(".bak")
   ```

4. **`_json_serialiser(obj)` static method:**
   ```python
   @staticmethod
   def _json_serialiser(obj):
       if isinstance(obj, Enum):
           return obj.value
       if dataclasses.is_dataclass(obj) and not isinstance(obj, type):
           return dataclasses.asdict(obj)
       raise TypeError(f"Object of type {type(obj)} is not JSON serializable")
   ```
   Import `from enum import Enum` at top of file.

5. **`write(self, state: dict[str, Any]) -> None`:**
   - Keep existing atomic write structure
   - Before `json.dump`, inject metadata into a copy of state:
     ```python
     now = datetime.now(timezone.utc)
     enriched = {
         **state,
         "_version": VERSION,
         "_written_at": now.timestamp(),
         "_written_at_iso": now.isoformat(),
     }
     ```
   - Use `enriched` for json.dump (not the original `state`)
   - Use `default=self._json_serialiser` instead of `default=str`
   - Before `os.replace(tmp, self.path)`, backup existing file if present:
     ```python
     if self.path.exists():
         shutil.copy2(self.path, self.backup_file)
     ```
   - Keep the outer `try/except Exception` with `logger.error`

6. **`read(self) -> dict[str, Any]`:**
   - Try primary file first (existing logic)
   - If primary fails (exception OR doesn't exist), try backup:
     ```python
     if self.backup_file.exists():
         with open(self.backup_file) as f:
             data = json.load(f)
         logger.warning("StateManager.read() loaded from backup file")
         return data
     ```
   - Return `{}` only if both fail
   - Strip metadata keys before returning? No — callers receive raw dict including `_version` etc. (main.py doesn't care about extra keys)

7. **`get_age_seconds(self) -> float`:**
   ```python
   def get_age_seconds(self) -> float:
       try:
           if self.path.exists():
               return time.time() - self.path.stat().st_mtime
       except Exception:
           pass
       return float("inf")
   ```
   Import `import time` (already likely present; add if missing).
